 Core Principles

Four pillars of a
trustworthy framework

Each design choice was made to maximise adoption by CACEIS leadership while respecting data sovereignty and regulatory constraints.

Data-Native

Every indicator is computed from datasets that already exist within CACEIS — IMR surveys, EAE evaluations, payroll data, absenteeism records, and training logs. No external benchmarks required to produce a first score.

IMR 2021–2025 EAE 5 766 rows Absenteeism detail P&L / FTE

Sovereign

The RAG layer runs on Mistral (French sovereign AI). Vector storage uses pgvector on-premises. No employee data ever leaves the CACEIS perimeter — fully compliant with GDPR and the Crédit Agricole Group's data governance policy.

Mistral RAG pgvector on-prem GDPR-ready CA Group compliant

Explainable

Every recommendation surfaces its source — the document, the data row, the formula. Managers can audit the reasoning chain end-to-end. No black box: the model cites before it concludes.

Source citations Formula audit trail Confidence scores Manager-readable

Actionable

Output is not a score — it is three concrete levers per BU. Each lever links a measured gap (engagement, training, absenteeism) to a financial impact estimate and a recommended HR programme already available at CACEIS.

 Methodology

How the model
is constructed

A four-step pipeline from raw HRIS exports to board-ready financial impact estimates — reproducible, versionable, and auditable at each stage.

Step 01
Data Ingestion & Normalisation
Six source files harmonised into a single anonymised employee-level fact table: EAE scores, IMR survey items, training hours, absenteeism days, compensation bands, and headcount. Deduplication, anonymisation, and schema validation applied.
Step 02
Composite Index Construction
Four dimensions weighted by a principal component analysis calibrated on CACEIS data: Engagement (Y), Training intensity (λ), Performance rating (β), and Financial productivity (P = NBI/FTE). Each dimension normalised 0–5.
Step 03
RAG-Augmented Analysis
Mistral runs over the document corpus (Bilans Sociaux, QVT accords, D&I barometers) using pgvector retrieval. The model contextualises the quantitative scores with qualitative programme data and surfaces gaps with cited evidence.
Step 04
Financial Impact Estimation
Each lever is converted to a €-impact range using CACEIS's own NBI/FTE ratio (€316k per FTE in 2025) and published absenteeism cost benchmarks. Confidence intervals reported alongside point estimates.
